# click dat

### An auto-clicker that clicks dat. whatever dat is.
<img width="1291" height="731" alt="image" src="https://github.com/user-attachments/assets/41ec8025-7430-4db8-99fb-bbb34c3120a2" />

A small, fast desktop auto-clicker with a simple GUI.

## Features

- Adjustable speed from 1 to 50 clicks per second
- `F6` to start/stop from anywhere (global hotkey)
- Minimizes on start so the first click never lands on the app
- Precise pacing that holds the target click rate

## Install

Cargo
```sh
cargo install click_dat
```

## Usage

Run `click_dat`, set your clicks per second, then press **Start** (or `F6`).
Press **Stop** (or `F6`) to halt clicking. The cursor is clicked wherever it
currently points.
